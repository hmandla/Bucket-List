🛠 Setup Help (click to expand)
  1. Create a Google Form with fields like Place, Location, Why, Added by.
  2. Click Send → <> Embed, copy the iframe URL (starts with https://docs.google.com/forms/d/e/) and replace YOUR_FORM_EMBED_URL above.
  3. Open the Form’s linked Google Sheet (Responses → View in Sheets).
  4. In the Sheet: File → Share → Publish to web → pick “Entire Document” → Publish.
  5. Copy the published link (it looks like https://docs.google.com/spreadsheets/d/e/2PACX-.../pubhtml) and replace YOUR_SHEET_EMBED_URL above.
  6. Commit this index.html to your GitHub repo and enable GitHub Pages (Settings → Pages → Source: deploy from main).
  7. Share your GitHub Pages URL with your partner 🎉
